# Sprint 4 — Data Quality и сверки

## Цель
Научиться находить расхождения и защищать качество витрин.

## Задачи
    1. Проверить дубликаты студентов по ИИН.
    2. Проверить оценки выше максимального балла.
    3. Проверить отсутствующие связи school/student/subject.
    4. Проверить удалённые оценки и их влияние на витрины.
    5. Сверить source и mart по количествам и суммам.

## Deliverables
- SQL checks
- отчёт по проблемам качества
- предложения по исправлению
